# GHF Express

GHF Express is a simple mobile app built with Ionic Vue 8 (Vue 3.x). It stores your
Gainesville Health & Fitness (GHF) membership ID and renders a barcode for quick entry.

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/us/app/ghf-express/id6737064794)
